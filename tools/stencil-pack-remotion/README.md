# Beyond Tattoo Stencil Pack Motion

1. Export a pack PNG from `/admin/stencil-pack-generator.php`.
2. Copy it to `public/stencil-pack.png`.
3. Run `npm install` and `npm run studio` for preview.
4. Run `npm run render` to render a six-second 1080×1350 MP4 with drifting smoke and foil shimmer.

The generator itself works on shared PHP hosting. This Remotion folder is optional for social-video exports on a local computer or Codex workspace.
