# Beyond OS 2.2.1 — Zak’s Kitchen Rush

## Added

- Playable `beyond-games/zaks-kitchen-rush.php` restaurant-management game.
- Timed customer queue with patience and service-rating systems.
- Haitian and American recipe sequences.
- Order combos and temporary Kitchen Rush speed boosts.
- Persistent local scores, achievements, coins and kitchen upgrades.
- Responsive desktop and mobile play modes with touch controls.
- Local demo bit$ achievement guardrail capped at 75 per day.

## Integrated

- Beyond Games publisher homepage now features Zak’s Kitchen Rush as playable build 002.
- Beyond Games catalogue metadata now launches the playable game directly.
- Beyond OS App Store card now uses a **Play now** action.
- Beyond OS homepage game showcase now promotes both Zak’s Kitchen Rush and Bit Runner.
- Existing privacy-conscious visitor analytics automatically cover the new route.

## Production safety

- No production `var/` files are included or replaced.
- The game uses browser-local progression and requires no database migration.
- Existing Beyond ID, wallet, analytics, database and configuration values remain untouched.
