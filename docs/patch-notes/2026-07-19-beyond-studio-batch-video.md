# Beyond Studio organization and batch video export

- Groups Studio navigation and dashboard tools by DailyBreath, Beyond French, Beyond Tattoo, and Shared Tools.
- Moves the stencil library, stencil pack generator, and tattoo publishing links into Beyond Tattoo.
- Adds compact mobile navigation styling for the Studio sidebar.
- Adds **Export all languages as one narrated MP4** to the Verse of the Day generator.
- Adds the same combined narrated-video export to the Beyond French generator.
- Records English, French, Spanish, Patois, and Kreyòl in sequence, pairing each language's displayed text with its matching narration voice.
- Restores the editor's original language, voice, date, and content after export.
- Falls back to WebM when the browser does not provide MP4 recording support.
