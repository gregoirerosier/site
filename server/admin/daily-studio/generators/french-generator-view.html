Français du Jour Generator

Build the daily four-language lesson and export it as an image or narrated short.

Patois and Kreyòl use the current server speaker when it supports those locales. Upload audio to override or supply a voice when it does not.
MP4 is used when supported; WebM is the fallback. When no API voice or uploaded file is available, the visual exports without audio instead of failing.
