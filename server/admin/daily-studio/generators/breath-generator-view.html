Verse of the Day Generator

Create a Daily Breath image, preview narration, and export a social-ready video.

English uses KJV. French and Spanish use their matching hardcoded scripture banks.
Preview uses a matching voice installed on this device. Narrated export uses the configured server voice for the selected language or an uploaded file.
